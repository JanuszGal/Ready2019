Exercise 16: Exporting settings as a .json file

1. In order to successfully export your settings. Please download the latest Intune PowerShell scripts from here (download and expand the ZIP file):

2. Start an elevated PowerShell session

3. Load the Azure AD Module. (Install-Module AzureAD)

4. Navigate to C:\PSHELL\Intune-scripts\DeviceConfiguration

5. Run the following script "C:\PSHELL\Intune-scripts\DeviceConfiguration\DeviceConfiguration_Export.ps1"

6. Authenticate to the Intune Tenant where you have your configuration

7. All of your Device Configurations will be downloaded

8. Send the contents of the Output folder to a compressed ZIP folder and upload to your Teams site or a OneDrive location for storage.

9. Start an elevated PowerShell session

10. Load the Azure AD Module. (Install-Module AzureAD)

11. Navigate to C:\PSHELL\Intune-scripts\DeviceConfiguration

12. Run the following script "C:\PSHELL\Intune-scripts\DeviceConfiguration\DeviceConfiguration_Import_FromJSON.ps1"

13. Authenticate to the Intune Tenant where you want to import your configuration

14. You will be asked to enter the configuration file for import enter "C:\PSHELL\Output\CustomDevicePolicy.json"

15. Log into or refresh your Intune Device Configuration session and verify you now see the profile. (note the profile will not be assigned.

16. Assign the profile to a well scoped group for production pilot testing.

17. Repeat process for each JSON file you want to import.
